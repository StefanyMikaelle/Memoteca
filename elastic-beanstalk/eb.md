# Elastic Beanstalk

## Testar a aplicação
  
  eb init memotecav2

## Iniciar a aplicação
  
  eb local run --port 5000

## Executar a aplicação

  eb local open

## Criar aplicação na AWS

  eb create memotecav2

## Status da aplicação

  eb status

## Abrir aplicação na AWS

  eb open

## Atualizar a aplicação na AWS

  eb deploy

## Remover recursos

  eb terminate --all --force
